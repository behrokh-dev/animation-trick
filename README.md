## ✨ About The Project

This is a minimal front-end project created to explore the power of CSS animations.  
There is no JavaScript involved — everything you see is achieved using pure CSS.

The goal of this project was to:
- Practice animation techniques
- Understand timing and transitions
- Improve visual creativity in UI design

---

## 🚀 Features

- 🎨 Pure CSS animations
- ⚡ Lightweight and fast
- 🧩 No JavaScript used
- 📱 Responsive behavior (basic)

---

## 🛠️ Built With

- HTML5  
- CSS3 (Animations, Transitions, Keyframes)

---

## 📚 What I Learned

- How to use `@keyframes` effectively
- Creating smooth transitions
- Controlling animation timing and delays
- Combining multiple animation effects
